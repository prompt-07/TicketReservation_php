## Instructions :

```
Step_1 - Download and launch xampp

Step_2 - Reachout to '\htdocs' and add the downloaded project to same

Step_3 - Launch phpMyadmin from 'localhost/' 

Step_4 - Create new database with name 'airline_reservation'

Step_5 - Move to 'Import' tab in newly created database and import the database file with '.sql' extension which is in '\Database File for import'

Step_6 - use url: 'http://localhost/rail-ticket-reservation/home_page.php' to launch the website.
```
